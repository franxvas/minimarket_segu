# Manual de usuario

1. Abra la URL local e inicie sesion con una cuenta ficticia autorizada.
2. Use el menu para abrir Productos, Clientes, Proveedores, Compras o Ventas.
3. En listas puede filtrar, ordenar, exportar y abrir un registro.
4. Para crear, pulse **Nuevo**, complete los obligatorios y **Grabar**.
5. En una compra agregue productos, cantidades y precios; guarde y pulse **Procesar** para sumar stock.
6. En una venta seleccione cliente, empleado, metodo y comprobante; agregue detalles, guarde y pulse **Procesar** para descontar stock.
7. Una transaccion procesada no puede afectar el inventario por segunda vez.
8. Para editar abra el registro, cambie datos y grabe. Para eliminar use **Eliminar**; las relaciones pueden impedir eliminar maestros utilizados.

Los mensajes debajo del formulario explican campos invalidos, falta de stock o una operacion ya procesada.
